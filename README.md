### Dependencies

Required:

Under GNU/Linux environment( Makefile is not tested on Windows or Mac OS )

1. GNU make

```sh
sudo apt-get update
sudo apt-get upgrade
sudo apt-get install make
sudo apt-get install graphviz
```
